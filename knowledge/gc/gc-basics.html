Gas Chromatograph (GC) – Basics

Category: Analyzer · GC · Fundamentals

Working Principle

Gas Chromatograph separates components of a gas sample based on retention time using a column and detector.

Main Components

  • Sample injection system
  • Column oven
  • Carrier gas system
  • Detector (FID, TCD, etc.)

Calibration

Calibration is performed using certified calibration gases. Zero and span verification should be done periodically.

Common Problems

  • Baseline drift
  • Poor peak separation
  • Retention time shift
  • Detector noise

Troubleshooting Tips

Check carrier gas purity, column condition, detector temperature stability, and leaks.
