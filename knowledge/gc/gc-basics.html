Gas Chromatograph (GC) – Basics

Gas Chromatography is an analytical technique used to separate and measure components of a gas mixture. In process analyzers, GC is widely used for refinery, petrochemical, gas processing, and environmental applications.

GC Working Principle – Basic Flow

A GC separates components based on their interaction with the column stationary phase. Each component exits the column at a different time (retention time) and is measured by the detector.

Main GC Components

A process GC analyzer consists of mechanical, chromatographic, and electronic sections. Understanding each block helps in troubleshooting analyzer problems.

Key Points for Technicians

  • GC separates components based on retention time.
  • Injection valve timing is critical for repeatability.
  • Column condition directly affects accuracy and response time.
  • Most GC issues originate from sampling problems.
