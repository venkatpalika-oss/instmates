Loop Check vs Bench Calibration

Loop check and bench calibration serve different purposes in a 4–20 mA system. Confusing them can lead to wrong conclusions, wasted time, and unnecessary transmitter replacement.

What is a Loop Check?

A loop check verifies the entire signal path from the field instrument to the control system.

Purpose

  • Confirm correct wiring
  • Verify PLC/DCS scaling
  • Check display and alarms
  • Ensure signal reaches control room

How It Is Done

  • Inject 4–20 mA using a loop calibrator
  • Observe value
