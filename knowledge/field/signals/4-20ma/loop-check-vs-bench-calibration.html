Loop Check vs Bench Calibration

Loop check and bench calibration serve different purposes in a 4–20 mA system. Confusing them can lead to wrong conclusions, wasted time, and unnecessary transmitter replacement.

What is a Loop Check?

A loop check verifies the entire signal path from the field instrument to the control system.

Purpose

  • Confirm correct wiring
  • Verify PLC/DCS scaling
  • Check display and alarms
  • Ensure signal reaches control room

How It Is Done

  • Inject 4–20 mA using a loop calibrator
  • Observe value at PLC/DCS
  • No process input required

A loop check does not verify sensor accuracy.

What is Bench Calibration?

Bench calibration verifies the instrument accuracy by applying a known physical input and measuring the output.

Purpose

  • Check sensor accuracy
  • Adjust zero and span
  • Generate calibration records
  • Meet QA / audit requirements

How It Is Done

  • Instrument isolated or removed from process
  • Known input applied (pressure, temperature, etc.)
  • Output measured directly

Bench calibration does not verify control room wiring or scaling.

Side-by-Side Comparison

Aspect Loop Check Bench Calibration
Checks wiring ✔ Yes ✖ No
Checks sensor accuracy ✖ No ✔ Yes
Requires process input ✖ No ✔ Yes
Checks PLC/DCS scaling ✔ Yes ✖ No
Typical timing Commissioning / troubleshooting Maintenance / QA

Common Field Mistakes

  • Assuming loop check confirms transmitter accuracy
  • Calibrating a transmitter with wrong loop wiring
  • Skipping loop check after instrument replacement

Field Rule

Always perform a loop check after wiring changes, and a bench calibration when accuracy is in question. One does not replace the other.
