4–20 mA Troubleshooting – Master Guide

This guide helps instrument technicians systematically diagnose 4–20 mA loop problems using field logic — not guesswork. Always measure current first before changing configuration.

Golden Rule Before Troubleshooting

  • Measure actual loop current with a multimeter
  • Do NOT rely only on DCS value
  • Confirm transmitter type (2-wire / 3-wire / 4-wire)
  • Verify loop power supply voltage

80% of loop issues are wiring, grounding, or power related.

Problem 1 – No Signal (0 mA)

  • Open loop (broken cable)
  • Blown fuse
  • Power supply failure
  • Loose terminal connection

Field Check Sequence

  1. Measure supply voltage at transmitter terminals
  2. Check continuity of loop wiring
  3. Inspect marshalling panel terminals
  4. Check intrinsic safety barrier (if installed)

Problem 2 – Fixed at 4 mA

4 mA represents minimum value (live zero).

  • Process variable at minimum
  • Sensor disconnected internally
  • Transmitter underrange fault
  • Configuration scaling mismatch

Field Logic

  • Simulate input if possible
  • Check sensor wiring
  • Review LRV configuration

Problem 3 – Fixed at 20 mA

  • Process at maximum range
  • Overrange condition
  • Short circuit in wiring
  • Incorrect scaling in DCS

Field Logic

  • Check URV configuration
  • Inspect for signal short
  • Compare with local indicator

Problem 4 – Unstable / Fluctuating Signal

  • Poor grounding
  • Shield not terminated correctly
  • Loose terminal screws
  • Electromagnetic interference
  • Power supply ripple

Field Checks

  • Ground shield at one side only
  • Check earth resistance
  • Separate signal cable from power cables
  • Verify stable 24V supply

Problem 5 – Reading Correct Locally but Wrong in DCS

  • DCS scaling error
  • Incorrect input card type (active/passive)
  • Ground loop between panels
  • Signal drop due to loop resistance

Field Logic

  • Measure current at DCS input
  • Verify LRV / URV in both transmitter and DCS
  • Check loop resistance calculation

Quick Fault Reference Table

  • 0 mA → Open circuit
  • ~4 mA steady → Underrange or minimum process
  • ~20 mA steady → Overrange or short
  • Random jumps → Shielding / grounding issue
  • Slow drift → Power or sensor instability

Advanced Checks (Experienced Technicians)

  • Loop resistance calculation (Ohm’s law)
  • Voltage drop across barrier
  • Check for multi-ground points
  • Inspect cable gland moisture ingress
  • Check intrinsic safety compliance

Final Field Rule

Never replace a transmitter before verifying: power, wiring, grounding, scaling, and loop continuity. Most “bad transmitters” are actually bad installations.
