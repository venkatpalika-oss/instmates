GC Sampling System – Deep Dive

Category: Analyzer · Gas Chromatograph · Sampling System

Why the Sampling System Is Critical

In real plant operation, the GC analyzer itself is rarely the problem. Most GC failures originate from the sampling system.

Any instability in pressure, temperature, flow, or cleanliness before the sample reaches the column will directly affect:

  • Retention time
  • Peak area and repeatability
  • Baseline stability
  • Calibration success

Typical GC Sampling Flow

Key Sampling System Components

1. Sample Probe

  • Correct insertion depth is critical
  • Plugged probes cause slow response and false stability
  • Improper location leads to non-representative samples

2. Sample Filters

  • Protects valves and columns from particulates
  • Partial choking causes pressure fluctuation
  • Fully choked filters cause zero or flat readings

3. Sample Lines

  • Should be as short as possible
  • Low points cause liquid accumulation
  • Improper material causes adsorption

4. Pressure Regulation

Sample pressure stability is more important than flow for GC repeatability.

  • Pressure fluctuation shifts retention time
  • Regulator creep causes peak area drift
  • Shared regulators create cross-interference

5. Sample Conditioning

  • Heaters prevent condensation
  • Coolers remove excess moisture
  • Poor conditioning causes negative peaks

Common Sampling System Failures

  • Condensation during night operation
  • Filter choking after process upset
  • Pressure regulator sticking
  • Leaks at compression fittings
  • Probe blockage from process carryover

Field Troubleshooting Sequence

  1. Verify sample pressure stability
  2. Inspect filters visually
  3. Check for cold spots and condensation
  4. Leak test all fittings
  5. Compare GC trends with process changes

If GC behavior changes with ambient temperature or time of day, the sampling system should be suspected first.
